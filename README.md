# Automação de Organização de Arquivos em Python

## 📌 Descrição
Script desenvolvido em Python para automatizar a organização de arquivos em pastas, separando por tipo de extensão.

## ⚙️ Funcionalidades
- Criação automática de pastas
- Organização de arquivos por extensão
- Automação de tarefas repetitivas
- Uso de lógica de programação e estruturas de controle

## 🛠️ Tecnologias Utilizadas
- Python 3
- Biblioteca os
- Biblioteca shutil
- Visual Studio Code

## ▶️ Como executar
1. Clone o repositório
2. Crie uma pasta chamada `arquivos`
3. Coloque arquivos de diferentes extensões dentro dela
4. Execute o comando:
   ```bash
   python organizador.py
